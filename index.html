Proyectos

QA Manual, Automation, and Web Development | ISTQB Test Web

Desarrollé una página web interactiva para un examen de opción múltiple de ISTQB, que corrige automáticamente las respuestas y muestra las puntuaciones en tiempo real. Al aplicar metodologías ágiles y BDD, lideré el proceso de aseguramiento de la calidad, que incluyó el diseño de pruebas, la ejecución manual y la implementación de un plan de pruebas de regresión utilizando Cypress y Cucumber para las versiones web y móvil.

JIRA, JavaScript, Cypress, Cucumber, BDD, HTML, CSS, GitHub, Visual Studio Code

QA Automation | Spinmatic

Diseñé y automatizé un plan de pruebas de regresión para la plataforma de juegos en línea de Spinmatic, utilizando Cucumber y Playwright en JavaScript. Las pruebas se ejecutaron a través de GitHub Actions, asegurando la integración continua y el aseguramiento de la calidad para la plataforma web.

JavaScript, Playwright, Cucumber, GitHub, Visual Studio Code

QA API Testing | JIRA

Desarrollé y ejecuté un plan de pruebas automatizadas para la funcionalidad de comentarios de JIRA, centrándome en el endpoint de comentarios. Utilizando Postman, validé los códigos de respuesta (éxito, no autorizado, no encontrado, error) y los resultados de varias solicitudes GET, POST, PUT y DELETE.

Postman, JIRA, API Testing
